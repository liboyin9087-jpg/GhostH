# 靈異連線 - 快速導覽

這個儲存庫包含兩個完整專案，提供不同的使用場景。

## 🎯 選擇你的起點

### 1️⃣ 想要可重用的 UI 組件？
👉 前往 **[optimized-project](./靈異連線_完整優化包_v3.2_Final/optimized-project/)**
- 11 個獨立的 React 組件
- 可集成到任何 React 專案
- 包含 AI 服務整合
- 適合：開發者、設計師、募資演示

### 2️⃣ 想要完整的遊戲演示？
👉 前往 **[spectrallink-demo](./靈異連線_完整優化包_v3.2_Final/spectrallink-demo/)**
- 可玩的互動式原型
- 完整音效與視覺效果
- 三個可探索場景
- 適合：玩家測試、展示、遊戲開發參考

## 🚀 5 分鐘體驗

### 體驗組件庫
```bash
cd 靈異連線_完整優化包_v3.2_Final/optimized-project
npm install
npm run dev
```

### 體驗遊戲演示
```bash
cd 靈異連線_完整優化包_v3.2_Final/spectrallink-demo
npm install
npm run dev
```

## 🎮 遊戲演示功能

- ✅ 三個可探索場景（B1 走廊、護理站、太平間）
- ✅ 互動式物件檢查系統
- ✅ 掃描器與能量場偵測
- ✅ VHS 風格視覺效果（掃描線、色差、雜訊）
- ✅ 完整音效系統（30+ 個音效檔案）
- ✅ 護符封印機制
- ✅ 成就與進度追蹤
- ✅ 回放系統（重看靈異現象）

## 🛠️ 開發者資源

### SpectralLink Demo 特有資源
- **音效庫**: `public/audio/` - 包含環境音、鬼魂、UI、VHS 等分類
- **圖片資源**: `public/images/` - 場景背景、熱點、UI 按鈕
- **物件數據**: `src/specpack/data/` - 互動物件與腳本
- **開發工具**: `_ghosth_addons/scripts/` - 驗證與打包工具
