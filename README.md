# 靈異連線：蝕骨杏林 (GhostH - Spectral Link)

> 台灣文化特色恐怖遊戲 - 1990年代廢棄醫院靈異探索體驗

## 📖 專案簡介

《靈異連線：蝕骨杏林》是一款融合台灣民俗文化的互動式恐怖遊戲原型，採用VHS監視器美學，讓玩家透過手電筒、掃描器與道教護符等工具，探索廢棄杏林醫院的詭異事件。

**技術棧**: React 18 + TypeScript + Vite  
**版本**: v4.0 Complete Edition

## ✨ 核心特色

- 🎮 **三個可探索場景** - B1走廊、護理站、太平間
- 🎵 **30+ 專業音效** - 完整的環境音、鬼魂音效、UI音效庫
- 📹 **VHS視覺效果** - 掃描線、色差、雜訊等懷舊風格
- 🎴 **台灣民俗元素** - 三魂七魄、道教符咒、祭改儀式
- 🏆 **遊戲系統** - 成就追蹤、線索收集、驚嚇導演

## 🚀 快速開始

```bash
# 進入專案目錄
cd 靈異連線_完整優化包_v3.2_Final/optimized-project

# 安裝依賴
npm install

# 啟動開發伺服器
npm run dev
```

遊戲將在瀏覽器中啟動（預設 http://localhost:5173）

## 🎮 遊戲操作

| 工具 | 功能 |
|------|------|
| 🔦 手電筒 | 探索黑暗區域 |
| 📡 掃描器 | 偵測靈異能量場 |
| 📹 回放系統 | 查看監視器記錄 |

| 📋 線索抽屜 | 查看收集的證據 |
