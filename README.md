## 🏃 快速開始

```bash
# 1. 進入專案目錄
cd 靈異連線_完整優化包_v3.2_Final/optimized-project

# 2. 安裝依賴
npm install

# 3. 設定環境變數（可選）
cp .env.example .env
# 編輯 .env 並填入你的 VITE_GEMINI_API_KEY

# 4. 啟動開發伺服器
npm run dev

# 5. 瀏覽器會自動開啟 http://localhost:5173
```

詳細步驟請參考：[靈異連線_完整優化包_v3.2_Final/QUICKSTART.md](./靈異連線_完整優化包_v3.2_Final/QUICKSTART.md)
