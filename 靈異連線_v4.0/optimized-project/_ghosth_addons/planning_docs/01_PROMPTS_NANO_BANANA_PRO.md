# Nano Banana Pro：CUTOUT / FOUND Prompt 模板（固定版）

## BASE（每張都要放最前面）
Nano Banana Pro. Photorealistic.

---

## CUTOUT 模板（透明去背 PNG）
SUBJECT: {物件名稱}, worn, dusty, slightly stained, realistic material imperfections, tiny scratches, hospital-use grime.
Camera: product shot, 3/4 angle, soft studio light, soft shadow.
Output: 1:1 square, isolated on transparent background (PNG alpha), clean cutout edge.

Negative: readable text, misspelled words, logos, watermark, UI/HUD, CGI, 3D render, overly clean new look, plastic toy look.

---

## FOUND 模板（場景尋獲）
SUBJECT: {物件名稱} in an abandoned Taiwan hospital (1990s architecture). First-person flashlight beam from {方位}, heavy vignette, VHS scanlines, analog noise, slight motion blur, subtle camera shake, humid dusty air, cinematic low-light.

Composition rules:
- 不要每張都桌上；可放地上、角落、遠景、半遮蔽、背後、抽屜裡、推車下
- 物件可部分被遮擋，增加「找得到但不明顯」的遊戲性

Negative: readable text, logos, watermark, UI/HUD, CGI, 3D render, clean modern hospital, bright daylight.

---

## 參數小提醒（避免偏綠）
- 若畫面整體偏綠太重：把 cast 改成 “slightly greenish cast” 或 “neutral cold cast”，並提高灰階對比（但不要變成霓虹綠夜視）
