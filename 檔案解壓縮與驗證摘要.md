- 💡 建議：如需高品質音效，需要額外錄製或購買授權音檔

### 2. 需要 API Key 設定 ⚠️
**必須設定才能使用 AI 功能**

- 需要：Google Gemini API Key
- 設定方式：
  1. 複製 `.env.example` 為 `.env`
  2. 填入 `VITE_GEMINI_API_KEY=你的金鑰`
- 取得位置：https://makersuite.google.com/app/apikey

### 3. Node.js 版本需求 ℹ️
**系統需求**

- Node.js >= 18.0.0
- npm >= 9.0.0

### 4. 依賴需要安裝 ℹ️
解壓縮後需要執行 `npm install` 安裝所有依賴套件

## 🚀 快速開始步驟

### 第一步：進入專案目錄
```bash
cd 靈異連線_完整優化包_v3.2_Final/optimized-project
```

### 第二步：安裝依賴
```bash
npm install
```

### 第三步：設定環境變數
```bash
# 複製環境變數範例檔案
cp .env.example .env

# 編輯 .env 檔案，填入你的 Gemini API Key
# VITE_GEMINI_API_KEY=你的實際金鑰
```

